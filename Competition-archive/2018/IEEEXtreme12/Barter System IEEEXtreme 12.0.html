    Barter System

    Time limit: 1000 ms
    Memory limit: 256 MB

    In a world where currency is not available, people fulfill their necessities by exchanging commodities among them. However, not all commodities are equal.

    Exchange rate of commodities AAA to BBB is defined as the number of units of BBB one can get with one unit of AAA.

    Given exchange rates of a set of pairs of commodities as input, answer a set of queries by finding the exchange rate of a specific pair.

    Standard input

    The first line of the input is an integer NNN which indicates the number of given exchange rates to follow. The next NNN lines consists of A,B,rA, B, rA,B,r triplets where AAA and BBB are the commodities and rrr is the exchange rate such that A=rB (mod 998244353)A = r \cdot B \ (\text{mod} \ 998244353)A=rB (mod 998244353).

    The next line contains another integer QQQ which represents the number of queries. The following QQQ lines consists of a pair of commodities KKK and LLL.

    Standard output

    For each of the QQQ queries you need to find rrr such that K=rLK = r \cdot LK=rL.

    It can be shown that rrr can be represented as XY\frac{X}{Y}YX, where XXX and YYY are coprime integers and X0 (mod 998244353)X \ne 0 \ (\text{mod} \ 998244353)X0 (mod 998244353). For each query print XY1 (mod 998244353)X \cdot Y^{-1} \ (\text{mod} \ 998244353)XY1 (mod 998244353).

    If the exchange rate can not be computed using the given information, print -1.

    Constraints and notes

    • 1N,Q21041 \leq N, Q \leq 2 \cdot 10^41N,Q2104
    • 1A,B501 \leq |A|, |B| \leq 501A,B50
    • 1r<9982443531 \leq r < 9982443531r<998244353
    • All exchange rates are consistent
